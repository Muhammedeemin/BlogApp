# Blog Frontend

Angular 14 ile geliştirilmiş blog uygulaması frontend'i.

## Kurulum

```bash
npm install
```

## Çalıştırma

```bash
npm start
```

Uygulama http://localhost:4200 adresinde çalışacaktır.

## Notlar

- Backend API'nin çalıştığından emin olun (https://localhost:7006)
- CORS ayarlarının backend'de yapıldığından emin olun
